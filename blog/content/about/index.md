# About This Learning Log

This blog documents my learning journey through **Go in 15 weeks**. It serves as a place to track progress, consolidate learnings, and reflect on my discoveries.

## Structure

- **Learning Log**: Periodic posts reflecting on what I've learned each day/week
- **Day Folders**: Hands-on exercises from the Go-in-15-weeks curriculum
- **Consolidated Notes**: Summary of key concepts from `learnt.md`

## Purpose

This learning log helps me:
- Document progress consistently
- Reflect on and consolidate new concepts
- Create a reference for future learning
- Track the journey from beginner to proficiency in Go
